### 首页设计理念

- **卡片式导航**：每个工具/资源以精致卡片呈现，悬停有微动效（subtle lift + shadow）
- **全局搜索栏**：顶部居中，支持快速检索所有工具和资料
- **极简 Hero 区域**：大标题 + 一句话介绍 + 微妙渐变背景

### 4.4 🔄 复习模式（Review）— 核心：间隔重复算法

#### SM-2 算法（改良版）实现逻辑

javascript

编辑

```
1// 改良 SM-2 间隔重复算法
2class SpacedRepetition {
3  constructor() {
4    this.defaultEase = 2.5;    // 默认难度系数
5    this.minEase = 1.3;        // 最低难度系数
6    this.intervals = [1, 3, 7, 14, 30, 60, 120]; // 基础间隔(天)
7  }
8
9  /**
10   * @param {Object} card - 单词卡片数据
11   * @param {number} quality - 用户评分 (0-5)
12   *   0: 完全不认识（黑色状态）
13   *   1: 有极模糊印象
14   *   2: 有模糊印象，但想不起来
15   *   3: 经过思考后想起（黄色状态）
16   *   4: 稍加思考即想起（绿色状态）
17   *   5: 立即想起，毫不费力（蓝色/掌握状态）
18   */
19  calculate(card, quality) {
20    let { easeFactor, interval, repetitions } = card.srs;
21    
22    if (quality < 3) {
23      // 未通过：重置学习进度
24      repetitions = 0;
25      interval = 1; // 明天重来
26    } else {
27      // 通过：递增间隔
28      if (repetitions === 0) {
29        interval = 1;
30      } else if (repetitions === 1) {
31        interval = 3;
32      } else {
33        interval = Math.round(interval * easeFactor);
34      }
35      repetitions += 1;
36    }
37
38    // 更新难度系数
39    easeFactor = easeFactor + (0.1 - (5 - quality) * (0.08 + (5 - quality) * 0.02));
40    easeFactor = Math.max(this.minEase, easeFactor);
41
42    // 计算下次复习日期
43    const nextReview = new Date();
44    nextReview.setDate(nextReview.getDate() + interval);
45
46    return {
47      easeFactor,
48      interval,
49      repetitions,
50      nextReview,
51      status: this.getStatus(quality)
52    };
53  }
54
55  getStatus(quality) {
56    const statusMap = {
57      0: 'new',        // 全新/完全不认识
58      1: 'learning',   // 学习中
59      2: 'learning',   // 学习中
60      3: 'review',     // 需巩固
61      4: 'familiar',   // 较熟悉
62      5: 'mastered'    // 已掌握
63    };
64    return statusMap[quality];
65  }
66}
```

## 十一、总结：核心设计哲学

表格

|原则|说明|
|---|---|
|**克制即高级**|颜色不超过3种主色，大量留白，让内容成为主角|
|**动效有温度**|每个微交互都传递反馈信息，而非纯装饰|
|**流程无摩擦**|打开 → 学习，不超过2次点击|
|**数据即动力**|可视化进步曲线 + 成就系统，让坚持有回报|
|**键盘即效率**|全键盘操作支持，让学生高效沉浸不中断|
